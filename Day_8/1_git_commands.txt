1. git remote add origin <ssh-link> 
    ( This command connects your local repo with your remote repo .Your repo will be represented by name "origin")

2. git remote -v
    (This command displays all the remotes connected to your local repo)

3. ssh-keygen -t ed25519 -C "your_email@gmail.com"
    (It generates a new ssh-key for your local(laptop/pc))


4. git push -u origin main
    (This command push the code to your remote code)
